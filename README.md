# EGinz 

A graphical tool to help making e-cigarette liquid.
![Alt text](img/eginz.png?raw=true "Title")

## Launch 

Use the shell script in ```compiled/``` or use the command ```julia gui.jl```

## Dependencies 

[Julia](https://julialang.org/)
[GTK](https://www.gtk.org/)
